# login-google-auth0

## Local Development

<ul>
<li>Fork Repository</li>
<li>git clone https://github.com/paul-nguyen96/login-google-auth0.git </li>

## Start Local Server

<ul>
<li>docker compose up --build</li>
<li>Access local server: http://localhost:3000/</li>
</ul>

## Close Local Server

<ul>
<li>CTRL + C</li>
or
<li>docker compose down</li>
</ul>
